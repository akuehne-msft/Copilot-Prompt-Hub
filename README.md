# Copilot Prompt Hub
A dynamic collection of prompts designed to boost your personal productivity and spark self‑discovery—learn about yourself while mastering Copilot’s full potential.

## How to Use
- Paste a prompt into Copilot and **add context** (goal, audience, data source).
- **Iterate**: ask Copilot to refine tone, length, or structure.
- **Personalize**: inject your constraints (timebox, format, stakeholders).
